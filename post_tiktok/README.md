# post_tiktok

Has some scripts to compute and graph some stats.

- `counts.py` prints the count of each node and edge present in the `intiktok` dataset.
- `coll_heatmap_tiktok.py` yields a heatmap of collaborations, segregated by the artists main tag.
- `coll_heatmap_tiktok_year.py` yields a heatmap of collaborations, segregated by the artists begin_year.
- `follows_heatmap_tiktok.py` yields a heatmap of TikTok follows, segregated by the artists main tag.
- `follows_heatmap_tiktok_year.py` yields a heatmap of TikTok follows, segregated by the artists begin_year.
